# git_test
实验git各种操作

-  git branch
    -  查看分支
-  git branch -r
    - 查看远程分支
-  git branch -a
    - 查看全部分支
-  git branch dev
    - 本地新建分支
-  git checkout dev
    - 切换分支
-  git push origin dev
    - 推送分支到远程仓库
-  git branch -d dev
    - 删除分支
